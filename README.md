### Clone my source in your computer
```
git clone https://github.com/jomo34/React_Native_Login.git
cd RNPractice1
npm install
```

### Start Metro
```
cd RNPractice1
npm start
```

### Run app
```
cd RNPractice1
react-native run-android
        or
react-native run-ios
```
